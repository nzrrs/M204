# M204 – Développement Front-End

Ce dépôt contient les ressources du module **M204 – Développement Front-End**, organisées en deux dossiers principaux :

## Structure du dépôt

- **/cours**  
  Contient les notions théoriques, les supports et les exemples vus en classe.

- **/tps**  
  Contient les travaux pratiques (TP), exercices et mini-projets associés aux chapitres du dossier **cours**.

## Utilisation

1. Ouvre le dossier **cours** pour revoir les notions théoriques du module.  
2. Consulte le dossier **tps** pour réaliser ou vérifier les exercices liés aux chapitres du **cours**.
