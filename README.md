# M204 – Développement Front-End

Ce dépôt contient les ressources du module **M204 – Dev Front-End**, organisé en deux dossiers principaux :

## Structure du dépôt

- **/cours**  
  Contient les notions théoriques, supports et exemples vus en classe.

- **/tps**  
  Contient les travaux pratiques (TP), exercices et mini-projets associés aux chapitres du dossier **cours**.

## Comment utiliser ce dépôt

1. Ouvre le dossier **cours** pour revoir les notions théoriques du module.
2. Consulte le dossier **tps** pour voir ou réaliser les exercices liés aux chapitres de **cours**.
